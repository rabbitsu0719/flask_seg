# flask_seg

Flask 기반 이미지 처리 및 세그멘테이션 실습을 위한 프로젝트입니다.  
과제 및 학습 목적의 코드들을 날짜별로 정리하여 관리하고 있습니다.

---

## 📖 프로젝트 개요

이 저장소는 **Python + Flask 환경에서 이미지 처리 및 세그멘테이션 관련 기능을 실습**하기 위해 구성된 프로젝트입니다.  
각 디렉토리는 과제 단위 또는 실습 날짜 기준으로 분리되어 있으며,  
Flask 서버 구성, 이미지 처리 로직, 간단한 웹 UI 연동 등을 포함합니다.

## ⚙️ 실행 환경

- Python 3.x
- Flask
- OpenCV
- 기타 라이브러리: `requirements.txt` 참고

### 가상환경 및 라이브러리 설치

```bash
python -m venv venv
source venv/bin/activate   # macOS / Linux
pip install -r requirements.txt
```
